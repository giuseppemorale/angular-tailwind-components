# Alert

`TailwindAlert` mostra un messaggio con severità (success, warning, danger, info), titolo opzionale e possibilità di chiusura.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindAlert],
  template: `
    <tailwind-alert
      severity="info"
      title="Information"
      message="This is an informational alert."
      [dismissible]="true"
    />`
})
export class ExampleComponent {}
```
