# Badge

`TailwindBadge` è un’etichetta compatta con varianti di colore, dimensione e opzione “dot”.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindBadge],
  template: `<tailwind-badge variant="primary" size="md">Badge</tailwind-badge>`
})
export class ExampleComponent {}
```
