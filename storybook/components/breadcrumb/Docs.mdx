# Breadcrumb

`TailwindBreadcrumb` mostra una gerarchia di link con supporto all’ultimo elemento come testo corrente.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindBreadcrumb],
  template: `
    <tailwind-breadcrumb
      ariaLabel="Breadcrumb"
      separator="/"
      [items]="[
        { label: 'Home', link: '#', icon: 'home' },
        { label: 'Section', href: '#' },
        { label: 'Current' }
      ]"
    />`
})
export class ExampleComponent {}
```
