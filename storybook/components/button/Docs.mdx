# Button

`TailwindButton` è il pulsante principale della libreria con colori, varianti solid/flat/outlined/ghost/text e dimensioni.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindButton],
  template: `
    <tailwind-button type="button" color="primary" kind="solid" size="md" [disabled]="false">
      Button
    </tailwind-button>`
})
export class ExampleComponent {}
```
