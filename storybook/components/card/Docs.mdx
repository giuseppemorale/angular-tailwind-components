# Card

`TailwindCard` è un contenitore con slot opzionali per header, corpo, immagine e footer.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindCard, TailwindButton],
  template: `
    <tailwind-card [elevated]="false" [hoverable]="false" [hasHeader]="true" [hasFooter]="true">
      <div tailwind-card-header><h3 class="font-semibold">Titolo</h3></div>
      <p>Contenuto della card.</p>
      <div tailwind-card-footer>
        <tailwind-button size="sm">OK</tailwind-button>
      </div>
    </tailwind-card>`
})
export class ExampleComponent {}
```
