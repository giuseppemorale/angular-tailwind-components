# Checkbox

`TailwindCheckbox` è un controllo checkbox integrato con i form reattivi (`ControlValueAccessor`).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindCheckbox],
  template: `
    <tailwind-checkbox
      label="Accept terms"
      size="md"
      [checked]="false"
    />`
})
export class ExampleComponent {}
```
