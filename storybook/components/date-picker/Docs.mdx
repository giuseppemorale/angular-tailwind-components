# Date Picker

`TailwindDatePicker` permette di selezionare una data tramite calendario; espone un valore stringa (es. `yyyy-MM-dd`) ed è un `ControlValueAccessor`.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindDatePicker],
  template: `
    <tailwind-date-picker
      label="Data di nascita"
      format="dd/MM/yyyy"
      [(value)]="date"
    />`
})
export class ExampleComponent {
  date: Date | null = null;
}
```

## Comportamento

Il componente segue un pattern **draft/apply**:

1. L'utente apre il calendario cliccando sul campo.
2. Naviga tra mesi e seleziona un giorno — la selezione viene mantenuta in uno stato *draft* senza ancora modificare il valore del form.
3. **Oggi** porta il calendario al mese corrente e preseleziona la data odierna nel draft.
4. **Applica** (o `Apply`) conferma il draft, aggiorna il valore e chiude il pannello.

Questo garantisce che il form venga aggiornato solo all'intenzione esplicita dell'utente.

## Lingua

Il componente supporta italiano (`it`) e inglese (`en`) tramite il token `TAILWIND_DATETIME_LANGUAGE`.

### Configurazione globale (consigliata)

Imposta il token una sola volta a livello di `ApplicationConfig`:

```typescript
// app.config.ts
import { ApplicationConfig } from '@angular/core';
import { TAILWIND_DATETIME_LANGUAGE } from 'angular-tailwind-components';

export const appConfig: ApplicationConfig = {
  providers: [
    { provide: TAILWIND_DATETIME_LANGUAGE, useValue: 'it' }
  ]
};
```

### Configurazione locale (per singolo componente)

```typescript
@Component({
  providers: [{ provide: TAILWIND_DATETIME_LANGUAGE, useValue: 'en' }]
})
export class MyComponent {}
```

## Formattazione della data

Il valore emesso è un oggetto `Date` (orario locale a mezzanotte, es. `new Date(2026, 4, 8)`).
La visualizzazione nel campo input usa il parametro `format` (default `dd/MM/yyyy`) tramite `formatDate` di Angular.
