# Date Picker

`TailwindDatePicker` permette di selezionare una data tramite calendario; espone un valore stringa (es. ISO) ed è un `ControlValueAccessor`.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindDatePicker],
  template: `
    <tailwind-date-picker
      label="Date of Birth"
      placeholder="Pick a date"
      format="yyyy-MM-dd"
      [(value)]="date"
    />`
})
export class ExampleComponent {
  date = '';
}
```
