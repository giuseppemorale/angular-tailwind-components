# Date time picker

`TailwindDateTimePicker` combina data e ora in un unico valore `TailwindDateTimeValue` (`date`, `time`).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindDateTimePicker],
  template: `
    <tailwind-datetime-picker
      label="Appointment Date & Time"
    />`
})
export class ExampleComponent {}
```
