# Divider

`TailwindDivider` è un separatore visivo riusabile: linea orizzontale o verticale, bordo solido o tratteggiato, opzionale etichetta centrata (solo orizzontale).

---

## Utilizzo

```typescript
import { Component } from '@angular/core';
import { TailwindDivider } from 'angular-tailwind-components';

@Component({
  imports: [TailwindDivider],
  template: `
    <p>Sezione A</p>
    <tailwind-divider />
    <p>Sezione B</p>
    <tailwind-divider orientation="vertical" variant="dashed" />
  `
})
export class ExampleComponent {}
```

In un contenitore **flex** a riga, `orientation="vertical"` produce una linea verticale che si allunga in altezza (`self-stretch` sul contenitore padre quando necessario).
