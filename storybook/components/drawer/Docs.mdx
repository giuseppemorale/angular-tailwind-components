# Drawer

`TailwindDrawer` è un pannello laterale con animazione, titolo e metodi `open()` / `close()`.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindDrawer, TailwindButton, TailwindInput],
  template: `
    <div>
      <tailwind-button (click)="drawer.open()">Open drawer</tailwind-button>
      <tailwind-drawer #drawer title="Drawer Title" position="right" [closeOnBackdrop]="true">
        <p>Contenuto</p>
        <tailwind-button size="sm" (click)="drawer.close()">Close</tailwind-button>
      </tailwind-drawer>
    </div>`
})
export class ExampleComponent {}
```
