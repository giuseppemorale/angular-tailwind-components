# Input OTP

`TailwindInputOtp` è un gruppo di campi a singola cifra per codici OTP / PIN, con incolla, navigazione da tastiera e integrazione `ControlValueAccessor`. Il comportamento è ispirato a [PrimeNG InputOtp](https://primeng.org/inputotp).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindInputOtp],
  template: `
    <tailwind-input-otp
      label="Verification code"
      [length]="6"
      [(value)]="code"
      separator="-"
      [separatorAfterIndex]="2"
      (completed)="onComplete($event)"
    />`
})
export class ExampleComponent {
  code = '';

  onComplete(full: string) {
    console.log('OTP:', full);
  }
}
```
