# Input

`TailwindInput` è un campo di input con etichetta, stati di errore e integrazione `ControlValueAccessor`.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindInput],
  template: `
    <tailwind-input
      label="Email"
      placeholder="Enter your email"
      type="email"
      size="md"
      [(value)]="email"
    />`
})
export class ExampleComponent {
  email = '';
}
```
