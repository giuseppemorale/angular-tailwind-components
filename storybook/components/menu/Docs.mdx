# Menu

Il toggle (es. bottone) resta **sibling** di `tailwind-menu`. Passa il `click` a `open($event)` così il menu si posiziona sotto l’elemento che ha emesso l’evento (`currentTarget`). Dopo la prima apertura con anchor, puoi chiamare anche `open()` senza argomenti (riusa l’ultimo anchor).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindMenu, TailwindButton],
  template: `
    <tailwind-button (click)="menu.open($event)">Open menu</tailwind-button>
    <tailwind-menu #menu [items]="items" align="left" (onSelect)="onSelect($event)" />`
})
export class ExampleComponent {
  items = [
    { label: 'Profile', value: 'profile' },
    { label: 'Sign out', value: 'signout' }
  ];
  onSelect(item: unknown) {}
}
```

Per usare un elemento DOM esplicito: `menu.open(buttonEl)` con un `HTMLElement`.
