# Message

`TailwindMessage` è un messaggio inline compatto con severità e contenuto proiettato.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindMessage],
  template: `
    <tailwind-message severity="info">
      This is an informational message.
    </tailwind-message>`
})
export class ExampleComponent {}
```
