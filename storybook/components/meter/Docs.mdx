# Meter

`TailwindMeter` mostra una barra **segmentata** (stile [PrimeNG MeterGroup](https://primeng.org/metergroup)): ogni segmento ha `label`, `value` e colore opzionale. I valori sono normalizzati rispetto a **`max`** (default 100).

---

## Utilizzo

```typescript
import { Component } from '@angular/core';
import { TailwindMeter, TailwindMeterSegment } from 'angular-tailwind-components';

@Component({
  imports: [TailwindMeter],
  template: `
    <tailwind-meter
      [segments]="segments"
      [max]="100"
      [showLabels]="true"
      size="md" />
  `
})
export class ExampleComponent {
  segments: TailwindMeterSegment[] = [
    { label: 'Apps', value: 40, variant: 'primary' },
    { label: 'Other', value: 60, variant: 'info' }
  ];
}
```

Il tipo `TailwindMeterSegment` è esportato dalla libreria (`label`, `value`, `variant?` con `TailwindSeverity | 'primary'`).
