# Notification

`TailwindNotification` è un banner con titolo, severità, azioni opzionali tramite proiezione contenuti.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindNotification],
  template: `
    <tailwind-notification severity="info" title="Information" [dismissible]="true">
      Corpo della notifica.
    </tailwind-notification>`
})
export class ExampleComponent {}
```
