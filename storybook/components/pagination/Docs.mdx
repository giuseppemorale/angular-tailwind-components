# Pagination

`TailwindPagination` gestisce la navigazione tra pagine con riepilogo testuale configurabile.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindPagination],
  template: `
    <tailwind-pagination
      [totalItems]="50"
      [pageSize]="10"
      [currentPage]="1"
      ariaLabel="Pagination"
      summary="Visualizzati {start}-{end} di {total}"
    />`
})
export class ExampleComponent {}
```
