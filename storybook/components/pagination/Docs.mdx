# Pagination

`TailwindPagination` gestisce la navigazione tra pagine con riepilogo testuale configurabile.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindPagination],
  template: `
    <tailwind-pagination
      [totalItems]="50"
      [pageSize]="10"
      [currentPage]="1"
      ariaLabel="Pagination"
      summary="Visualizzati {start}-{end} di {total}"
    />`
})
export class ExampleComponent {}
```

### Default globale: `TAILWIND_PAGINATION_SUMMARY`

Il token **`TAILWIND_PAGINATION_SUMMARY`** (`InjectionToken<string>`) imposta il **testo predefinito** dell’input **`summary`** su tutte le **`tailwind-pagination`** dell’albero che ricevono quel provider, quando **`summary` non è passato** dal template. Se il token **non** è fornito, il default è **`'Showing {start}-{end} of {total}'`**. I placeholder **`{start}`**, **`{end}`**, **`{total}`** hanno lo stesso significato dell’input **`summary`**.

Per non ripetere la stringa su ogni istanza, registra il provider **una volta** a livello applicazione (stesso livello di router, HTTP client, ecc.).

```typescript
import { TAILWIND_PAGINATION_SUMMARY } from 'angular-tailwind-components';

bootstrapApplication(AppComponent, {
  providers: [
    { provide: TAILWIND_PAGINATION_SUMMARY, useValue: 'Visualizzati {start}-{end} di {total}' }
  ]
});
```

Se su una singola istanza imposti **`[summary]="..."`**, quel valore **prevale** sul default del token.
