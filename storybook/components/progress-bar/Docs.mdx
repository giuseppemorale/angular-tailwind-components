# Progress bar

`TailwindProgressBar` visualizza avanzamento percentuale, modo indeterminato e opzioni estetiche.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindProgressBar],
  template: `
    <tailwind-progress-bar
      [value]="65"
      label="Upload progress"
      [showLabel]="true"
      [showValue]="true"
      variant="primary"
      size="md"
      [indeterminate]="false"
      [striped]="false"
    />`
})
export class ExampleComponent {}
```
