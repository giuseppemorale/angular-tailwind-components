# Radio group

`TailwindRadioGroup` raggruppa opzioni radio con layout verticale o orizzontale e integrazione form.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindRadioGroup],
  template: `
    <tailwind-radio-group
      label="Plan"
      [options]="options"
      size="md"
      orientation="vertical"
      [(value)]="selected"
    />`
})
export class ExampleComponent {
  selected = 'free';
  options = [
    { value: 'free', label: 'Free' },
    { value: 'pro', label: 'Pro' }
  ];
}
```
