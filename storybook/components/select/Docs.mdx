# Select

`TailwindSelect` è un dropdown custom in stile Tailwind con etichetta, opzioni, navigazione da tastiera, selezione multipla opzionale e stati di errore (`ControlValueAccessor`).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindSelect],
  template: `
    <tailwind-select
      label="Framework"
      placeholder="Choose a framework"
      [options]="options"
      size="md"
      [(value)]="value"
    />`
})
export class ExampleComponent {
  value = '';
  options = [
    { value: 'angular', label: 'Angular' },
    { value: 'react',   label: 'React', disabled: true },
  ];
}
```

## Selezione multipla

Con `[multiple]="true"` il valore è un array `T[]` (vuoto se nulla è selezionato). Il trigger mostra le label selezionate come chip; il pannello resta aperto mentre si aggiungono o rimuovono voci.

```typescript
@Component({
  imports: [TailwindSelect],
  template: `
    <tailwind-select
      label="Framework"
      placeholder="Scegli uno o più framework"
      [options]="options"
      [multiple]="true"
      [(value)]="value"
    />`
})
export class ExampleComponent {
  value: string[] = [];
  options = [
    { value: 'angular', label: 'Angular' },
    { value: 'react', label: 'React' },
  ];
}
```
