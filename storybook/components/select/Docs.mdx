# Select

`TailwindSelect` è una select nativa con etichetta, opzioni e stati di errore (`ControlValueAccessor`).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindSelect],
  template: `
    <tailwind-select
      label="Framework"
      placeholder="Choose a framework"
      [options]="options"
      size="md"
      [(value)]="value"
    />`
})
export class ExampleComponent {
  value = '';
  options = [{ value: 'angular', label: 'Angular' }];
}
```
