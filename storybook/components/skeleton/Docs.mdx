# Skeleton

`TailwindSkeleton` mostra placeholder animati per testo, cerchio, rettangolo o bordi arrotondati.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindSkeleton],
  template: `
    <tailwind-skeleton variant="text" width="100%" height="2rem" />`
})
export class ExampleComponent {}
```
