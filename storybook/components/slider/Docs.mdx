# Slider

`TailwindSlider` è uno slider accessibile (trascinamento, tastiera, ARIA), ispirato a [PrimeNG Slider](https://primeng.org/slider). Supporta valore singolo o **range** a due thumb, layout orizzontale/verticale, step e tick opzionali.

Implementa **`ControlValueAccessor`**: il valore è `number` oppure `[number, number]` quando `range` è `true` (tipo esportato `TailwindSliderValue`).

---

## Utilizzo

```typescript
import { Component } from '@angular/core';
import { FormsModule } from '@angular/forms';
import { TailwindSlider } from 'angular-tailwind-components';

@Component({
  imports: [FormsModule, TailwindSlider],
  template: `
    <tailwind-slider [(ngModel)]="volume" [min]="0" [max]="100" [step]="1" />
    <tailwind-slider [(ngModel)]="window" [range]="true" [min]="0" [max]="24" [step]="1" />
  `
})
export class ExampleComponent {
  volume = 50;
  window: [number, number] = [8, 18];
}
```

Disabilitazione: attributo **`disabled`** sul componente oppure stato disabilitato del form (`setDisabledState`).
