# Spinner

`TailwindSpinner` indica caricamento con SVG animato, colore e orientamento label.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindSpinner],
  template: `
    <tailwind-spinner
      size="md"
      color="primary"
      label=""
      ariaLabel="Loading"
      orientation="horizontal"
    />`
})
export class ExampleComponent {}
```
