# Stepper

`TailwindStepper` organizza contenuti in passi; ogni passo è un `tailwind-step` con etichetta e descrizione.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindStepper, TailwindStep, TailwindInput],
  template: `
    <tailwind-stepper>
      <tailwind-step label="Account" description="Create your account">
        <tailwind-input label="Email" />
      </tailwind-step>
      <tailwind-step label="Done" description="Finish">
        <p>Complete.</p>
      </tailwind-step>
    </tailwind-stepper>`
})
export class ExampleComponent {}
```
