# Table

`TailwindTable` gestisce contenitore, ordinamento e paginazione client-side su `data`. L’header è proiettato (`<thead>`). Il corpo si definisce con **`tailwindTableRow`** sulla riga template (vedi sotto).

---

## Righe (`tailwindTableRow`)

Il componente ripete il template per ogni riga di `displayedData()`. La forma consigliata è **direttiva strutturale su `<tbody>`** (la `*` è necessaria affinché Angular espanda il template con `let row`):

```html
<tbody *tailwindTableRow="let row">
  <tr>
    <td>{{ row['name'] }}</td>
  </tr>
</tbody>
```

Ogni iterazione produce un `<tbody>` separato (consentito in HTML5). Per righe zebrate, selezione e click, il contesto del template espone anche `stripedRow`, `selected`, `selectable`, `toggleRow` (vedi story).

**Alternativa legacy:** solo celle, senza `<tbody>/<tr>` nel template:

```html
<ng-template tailwindTableRow let-row>
  <td>...</td>
</ng-template>
```

---

## Intestazioni ordinabili (`tailwindSortHeader`)

Solo le colonne che devono ordinare usano la direttiva su **`<th>`** (testo dell’intestazione come contenuto dell’elemento, senza `button`). **`tailwind-table`** intercetta click e tastiera (delegation) e applica l’ordinamento; non servono riferimenti (`#tbl`) né input sulla tabella.

---

## Utilizzo

Aspetto predefinito di **`<thead>`**, **`<th>`** e **`<td>`** (sfondo header, bordi, padding, tipografia, hover riga) è definito in **`table.component.css`** — nel template servono solo struttura semantica e direttive (`tailwindSortHeader`, `tailwindTableRow`).

```typescript
@Component({
  imports: [TailwindTable, TailwindTableRowDirective, TailwindSortHeaderDirective],
  template: `
    <tailwind-table [data]="rows" [paginated]="true" [pagination]="pagination" [emptyColspan]="2">
      <thead>
        <tr>
          <th scope="col" tailwindSortHeader sortKey="name">Nome</th>
          <th scope="col">Email</th>
        </tr>
      </thead>
      <tbody *tailwindTableRow="let row">
        <tr>
          <td>{{ row['name'] }}</td>
          <td>{{ row['email'] }}</td>
        </tr>
      </tbody>
    </tailwind-table>
  `
})
export class ExampleComponent {
  rows = [{ name: 'Alice', email: 'a@example.com' }];
  pagination = { totalItems: 1, pageSize: 10, currentPage: 1 };
}
```

## Metodo `sort(key)`

Disponibile su `TailwindTable` per ordinare da codice; nelle intestazioni usare `tailwindSortHeader`.
