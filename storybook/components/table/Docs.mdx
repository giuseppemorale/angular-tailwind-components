# Table

`TailwindTable` mostra dati tabellari con ordinamento, selezione righe, caricamento e paginazione integrata.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTable],
  template: `
    <tailwind-table
      [columns]="columns"
      [data]="rows"
      [selectable]="false"
      [striped]="false"
      [loading]="false"
      emptyMessage="Nessun dato"
      [paginated]="true"
      [pageSize]="10"
      summary="Visualizzati {start} - {end} di {total}"
    />`
})
export class ExampleComponent {
  columns = [{ key: 'name', label: 'Name', sortable: true }];
  rows = [{ name: 'Alice' }];
}
```
