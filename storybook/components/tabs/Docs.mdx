# Tabs

`TailwindTabGroup` e `TailwindTab` definiscono schede con contenuto proiettato e indice attivo (model).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTabGroup, TailwindTab],
  template: `
    <tailwind-tab-group ariaLabel="Example" [scrollable]="false" [(activeIndex)]="index">
      <tailwind-tab label="Overview">Overview content</tailwind-tab>
      <tailwind-tab label="Details">Details content</tailwind-tab>
    </tailwind-tab-group>`
})
export class ExampleComponent {
  index = 0;
}
```
