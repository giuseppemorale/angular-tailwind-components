# Tag

`TailwindTag` è un’etichetta stile “pill” uppercase per stati e categorie.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTag],
  template: `<tailwind-tag variant="neutral">Label</tailwind-tag>`
})
export class ExampleComponent {}
```
