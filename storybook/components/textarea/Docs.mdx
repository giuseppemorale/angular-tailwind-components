# Textarea

`TailwindTextarea` è un campo multi‑riga con etichetta, stati di errore, opzioni di ridimensionamento e integrazione `ControlValueAccessor` (template-driven e reactive forms).

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTextarea],
  template: `
    <tailwind-textarea
      label="Descrizione"
      placeholder="Inserisci il testo"
      [rows]="5"
      resize="vertical"
      size="md"
      [(value)]="description"
    />`
})
export class ExampleComponent {
  description = '';
}
```
