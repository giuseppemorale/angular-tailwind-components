# Time picker

`TailwindTimePicker` è un selettore di orario (`ControlValueAccessor`) con valore stringa nel formato **`HH:mm`** (es. `"14:30"`).

Mostra un unico campo readonly con un'icona orologio; al click si apre un pannello con **due `<select>`** (ore 00–23, minuti 00–59) e due pulsanti:

- **Adesso / Now** — imposta la bozza all'ora corrente
- **Applica / Apply** — conferma la selezione, aggiorna il valore e chiude il pannello

Chiudere il pannello con un secondo click sull'input annulla le modifiche alla bozza.

---

## Lingua dell&apos;interfaccia

Placeholder, etichette dei pulsanti (**Adesso / Now**, **Applica / Apply**) dipendono dal token **`TAILWIND_DATETIME_LANGUAGE`**.

Valori supportati: **`'it'`** o **`'en'`**. Se il token non è fornito il componente assume **`'it'`**.

### Configurazione consigliata: provider globale

**Applicazione standalone** (`app.config.ts`):

```typescript
import { ApplicationConfig } from '@angular/core';
import { TAILWIND_DATETIME_LANGUAGE } from 'angular-tailwind-components';

export const appConfig: ApplicationConfig = {
  providers: [
    { provide: TAILWIND_DATETIME_LANGUAGE, useValue: 'it' }
    // …altri provider
  ]
};
```

**Bootstrap diretto** (`main.ts`) o **modulo radice** (`AppModule`): stesso oggetto `{ provide: TAILWIND_DATETIME_LANGUAGE, useValue: 'it' }` nell&apos;array `providers`.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTimePicker],
  template: `
    <tailwind-time-picker label="Meeting Time" [(value)]="time" />`
})
export class ExampleComponent {
  time = '09:00';
}
```

Con reactive forms:

```typescript
@Component({
  imports: [TailwindTimePicker, ReactiveFormsModule],
  template: `
    <tailwind-time-picker formControlName="meetingTime" label="Orario riunione" />`
})
export class ExampleComponent {}
```
