# Time picker

`TailwindTimePicker` è un campo ora (`ControlValueAccessor`) con valore stringa.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTimePicker],
  template: `
    <tailwind-time-picker label="Meeting Time" [(value)]="time" />`
})
export class ExampleComponent {
  time = '';
}
```
