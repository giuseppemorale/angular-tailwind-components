# Title

`TailwindTitle` rende un titolo semantico (`h1`–`h6`) con testo obbligatorio e, opzionalmente, un’icona Solar Line Duotone **prima** del testo.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTitle],
  template: `
    <tailwind-title titleTag="h1" text="Dashboard" class="text-primary-700" />
    <tailwind-title titleTag="h2" text="Impostazioni" icon="settings" />
  `
})
export class ExampleComponent {}
```
