# Toast

I toast sono gestiti da `TailwindToastService` e resi dal componente `<tailwind-toast />`. Va incluso **una sola volta** nel template del componente radice dell‚Äôapp (ad es. `AppComponent`), cos√¨ i toast sono disponibili ovunque.

---

## Utilizzo

### 1 ‚Äî Aggiungi il contenitore nell‚Äô`AppComponent`

```typescript
import { Component } from '@angular/core';
import { TailwindToast } from 'angular-tailwind-components';

@Component({
  selector: 'app-root',
  imports: [TailwindToast /* ‚Ä¶RouterOutlet, ecc. */],
  template: `
    <!-- contenuto dell‚Äôapp -->
    <tailwind-toast />
  `
})
export class AppComponent {}
```

### 2 ‚Äî Mostra i toast dal servizio

Da qualsiasi componente inietta `TailwindToastService` e chiama i metodi (`success`, `warning`, ecc.).

```typescript
import { Component, inject } from '@angular/core';
import { TailwindToastService, TailwindButton } from 'angular-tailwind-components';

@Component({
  imports: [TailwindButton],
  template: `
    <tailwind-button (click)="toast.success('Saved!', 'Success')">Success</tailwind-button>
  `
})
export class ExampleComponent {
  toast = inject(TailwindToastService);
}
```
