# Toggle

`TailwindToggle` è un interruttore on/off con `ControlValueAccessor`.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindToggle],
  template: `
    <tailwind-toggle
      label="Enable notifications"
      ariaLabel="Toggle notifications"
      size="md"
      [(checked)]="on"
    />`
})
export class ExampleComponent {
  on = false;
}
```
