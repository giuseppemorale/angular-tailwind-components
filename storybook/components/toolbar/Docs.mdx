# Toolbar

`TailwindToolbar` è una barra applicativa orizzontale o un rail laterale verticale. Il **menu** centrale è un array `TailwindMenuItem[]`; logo e azioni finali restano **content projection** (`tailwind-toolbar-logo`, `tailwind-toolbar-end`).

---

## Utilizzo

```typescript
import { Component } from '@angular/core';
import { TailwindButton, TailwindMenuItem, TailwindToolbar } from 'angular-tailwind-components';

@Component({
  imports: [TailwindToolbar, TailwindButton],
  template: `
    <tailwind-toolbar [menu]="menu" (onMenuSelect)="onNav($event)">
      <tailwind-toolbar-logo>Brand</tailwind-toolbar-logo>
      <tailwind-toolbar-end>
        <tailwind-button size="sm">Account</tailwind-button>
      </tailwind-toolbar-end>
    </tailwind-toolbar>
  `
})
export class ExampleComponent {
  menu: TailwindMenuItem[] = [
    { label: 'Home', value: 'home' },
    { label: 'Docs', value: 'docs' }
  ];

  onNav(item: TailwindMenuItem) {
    console.log(item.value);
  }
}
```

Con **orientation** `vertical`, le voci del menu sono in colonna; con `divider: true` su un elemento si ottiene un separatore orizzontale. In orizzontale il divider è una linea verticale tra le voci.
