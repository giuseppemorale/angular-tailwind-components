# Tooltip

`TailwindTooltipDirective` (`selector: '[tooltip]'`) mostra un overlay al passaggio del mouse o al focus sull’elemento host.

---

## Utilizzo

```typescript
@Component({
  imports: [TailwindTooltipDirective, TailwindButton],
  template: `
    <tailwind-button [tooltip]="'Hello'" [tooltipPosition]="'top'">Hover me</tailwind-button>`
})
export class ExampleComponent {}
```
