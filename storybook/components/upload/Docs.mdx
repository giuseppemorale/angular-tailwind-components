# Upload

`TailwindUpload` consente di scegliere file tramite **area drag‑and‑drop** o **pulsante** che apre il file picker nativo. Il valore per i form è una **data URL** (`data:<tipo>;base64,...`), leggibile ad esempio su `<img [src]>`. Per uso senza form model, usa l’output **`filesSelected`** sui `File` grezzi.

---

## Utilizzo

### Form / `[(value)]` (base64 come data URL)

```typescript
@Component({
  imports: [TailwindUpload, ReactiveFormsModule],
  template: `
    <form [formGroup]="form">
      <tailwind-upload
        label="Avatar"
        variant="area"
        accept="image/*"
        formControlName="avatarDataUrl"
      />
    </form>
  `
})
export class ExampleComponent {
  form = new FormGroup({
    avatarDataUrl: new FormControl('')
  });
}
```

### Solo output (`File[]`)

```typescript
@Component({
  imports: [TailwindUpload],
  template: `
    <tailwind-upload
      variant="button"
      buttonLabel="Scegli file"
      [multiple]="true"
      (filesSelected)="onFiles($event)"
    />
  `
})
export class ExampleComponent {
  onFiles(files: File[]) {
    /* upload HTTP, ecc. — senza aggiornare il FormControl */
  }
}
```

Con **`multiple`**, il modello / `FormControl` contiene la data URL **solo del primo file**; tutti i file scelti sono comunque emessi da **`filesSelected`**.
