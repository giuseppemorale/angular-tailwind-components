# Modal

`TailwindModal` è un overlay dialog che supporta due modalità d'uso: **dichiarativa** (template-driven) e **programmatica** (service-driven con risultato tipizzato).

---

## Utilizzo programmatico (via servizio)

Chiama `TailwindModalService.open()` da qualsiasi componente. La promise si risolve con il valore passato a `close()` oppure `undefined` se la modale viene chiusa senza risultato.

### 1 — Crea il componente modale

Il componente struttura il proprio contenuto con tre slot: `tailwind-modal-title`, `tailwind-modal-content`, `tailwind-modal-footer`. Inietta `TailwindModalRef<R>` per chiudere la modale con un risultato tipizzato.

```typescript
import { Component, inject } from '@angular/core';
import { TailwindButton, TailwindModal } from 'angular-tailwind-components';

@Component({
  imports: [TailwindModal],
  template: `
    <tailwind-modal-title>Conferma azione</tailwind-modal-title>

    <tailwind-modal-content>
      <p class="text-surface-700">L'operazione non può essere annullata. Vuoi procedere?</p>
    </tailwind-modal-content>

    <tailwind-modal-footer>
      <div class="flex justify-end gap-3">
        <tailwind-button color="secondary" kind="outlined" (click)="ref.dismiss()">
          Annulla
        </tailwind-button>
        <tailwind-button color="primary" (click)="ref.close(true)">
          Conferma
        </tailwind-button>
      </div>
    </tailwind-modal-footer>
  `
})
export class ConfirmModalComponent {
  readonly ref = inject<TailwindModalRef<boolean>>(TailwindModalRef);
}
```

### 2 — Apri la modale

```typescript
import { Component, inject } from '@angular/core';
import { TailwindModalService } from 'angular-tailwind-components';
import { ConfirmModalComponent } from './confirm-modal.component';

@Component({ ... })
export class MyPageComponent {
  private modalService = inject(TailwindModalService);

  async deleteItem(): Promise<void> {
    const confirmed = await this.modalService.open<boolean>(ConfirmModalComponent, {
      size: 'md',
      showCloseButton: true
    });

    if (confirmed) {
      // procedi con l'eliminazione
    }
  }
}
```

---

## Slot components

Il servizio distribuisce automaticamente i figli del componente nei tre slot di `TailwindModal`.

- **TailwindModalTitle** - `tailwind-modal-title` - Header — all'interno dell'`<h2>`
- **TailwindModalContent** - `tailwind-modal-content` - Body — `px-6 py-5`
- **TailwindModalFooter** - `tailwind-modal-footer` - Footer — `px-6 py-4 border-t`

> Tutti e tre i componenti vanno importati nel componente modale che li usa.

---

## TailwindModalRef\<R\>

Iniettato automaticamente nel componente modale dal servizio.

```typescript
const ref = inject<TailwindModalRef<boolean>>(TailwindModalRef);

ref.close(true);   // risolve la promise con `true`
ref.close();       // risolve la promise con `undefined`
```

Metodi:
- **close(result: R)** - Chiude la modale e risolve la promise con il valore dato

## Passare dati alla modale

Usa `config.data` per iniettare dati nel componente modale via `TAILWIND_MODAL_DATA`.

```typescript
// Componente modale
import { inject } from '@angular/core';
import { TAILWIND_MODAL_DATA, TailwindModalRef } from 'angular-tailwind-components';

interface DeleteData { itemName: string }

@Component({
  imports: [TailwindButton, TailwindModal],
  template: `
    <tailwind-modal-title>Elimina elemento</tailwind-modal-title>
    <tailwind-modal-content>
      <p>Stai per eliminare <strong>{{ data.itemName }}</strong>.</p>
    </tailwind-modal-content>
    <tailwind-modal-footer>
      <div class="flex justify-end gap-3">
        <tailwind-button kind="outlined" (click)="ref.dismiss()">Annulla</tailwind-button>
        <tailwind-button color="danger" (click)="ref.close(true)">Elimina</tailwind-button>
      </div>
    </tailwind-modal-footer>
  `
})
export class DeleteModalComponent {
  ref  = inject<TailwindModalRef<boolean>>(TailwindModalRef);
  data = inject<DeleteData>(TAILWIND_MODAL_DATA);
}

// Chiamante
const deleted = await this.modalService.open<boolean, { itemName: string }>(DeleteModalComponent, {
  data: { itemName: 'Fattura #1042' }
});
```

## Configurazione

```typescript
interface TailwindModalConfig<D = unknown> {
  data?:            D;            // Dati iniettati via TAILWIND_MODAL_DATA
  size?:            TailwindSize; // 'xs' | 'sm' | 'md' | 'lg' | 'xl'  (default: 'md')
  showCloseButton?: boolean;      // Mostra il pulsante × (default: true)
  closeOnBackdrop?: boolean;      // Chiude cliccando fuori (default: true)
  closeOnEscape?:   boolean;      // Chiude con Escape (default: true)
}
```

Size | Larghezza massima

- `xs` | `max-w-sm` (384px) |
- `sm` | `max-w-md` (448px) |
- `md` | `max-w-lg` (512px) — default
- `lg` | `max-w-2xl` (672px)
- `xl` | `max-w-4xl` (896px)

---

## Aperture multiple

Ogni chiamata a `open()` crea un injector isolato con la propria istanza di `TailwindModalRef`. È quindi possibile aprire più modali simultaneamente — ogni componente riceverà il suo `TailwindModalRef` specifico, senza conflitti.
