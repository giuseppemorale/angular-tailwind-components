# Icons

Le icone sono **[Heroicons](https://heroicons.com/outline)** in variante **outline** (24├ù24), licenza **MIT** (Tailwind Labs). Ogni SVG ├¿ servito con **larghezza e altezza intrinseche 64├ù64** e `viewBox` 24├ù24 cos├¼ si scalano da **16 a 64 px** tramite lÔÇÖattributo **`size`** del componente.

## Uso in TypeScript

Importa il tipo e lÔÇÖarray dei nomi dalla libreria:


Per ogni icona la **stringa da usare** ├¿ il nome in **kebab-case** come su [heroicons.com/outline](https://heroicons.com/outline), ad esempio `'bell'` oppure `'cog-6-tooth'`.

## Componente `tailwind-icon`

Per mostrare unÔÇÖicona usa **`TailwindIcon`** con:

- **`icon`**: `TailwindHeroicon` (obbligatorio)
- **`size`**: numero **pixel** per larghezza e altezza (**16ÔÇô64**, valori fuori range vengono clampati); se omesso usa il token **`TAILWIND_ICON_SIZE`** se presente, altrimenti **24**

Il componente imposta **`src`** su **`/tailwind-icons/<nome>.svg`** (URL dalla root del sito che serve lÔÇÖapp). Le SVG non sono incorporate nel bundle JS: il browser le scarica come risorse statiche.

```typescript

@Component({
  imports: [TailwindIcon],
  template: `<tailwind-icon icon="bell" [size]="16" />`
})
export class ExampleComponent {}
```

## Build della libreria e pacchetto npm

Prima di **`npm run build`** viene eseguito **`node scripts/copy-icons.js sync`**, che rigenera **`public/tailwind-icons/*.svg`** e **`models/icons.ts`** da `@iconify-json/heroicons`. Subito dopo la compilazione, **`node scripts/copy-icons.js dist`** copia gli SVG nel pacchetto pubblicato in **`tailwind-icons/`** (accanto a `fesm2022`, ecc.).

## App che usa la libreria (build di produzione)

## Dimensione predefinita: `TAILWIND_ICON_SIZE`

Il token **`TAILWIND_ICON_SIZE`** (`InjectionToken<number>`) imposta il **default** di **`size`** su tutte le **`tailwind-icon`** dellÔÇÖalbero che ricevono quel provider. Se il token **non** ├¿ fornito, il default ├¿ **24** (px).

```typescript
import { bootstrapApplication } from '@angular/platform-browser';
import { TAILWIND_ICON_SIZE } from 'angular-tailwind-components';
import { AppComponent } from './app.component';

bootstrapApplication(AppComponent, {
  providers: [{ provide: TAILWIND_ICON_SIZE, useValue: 20 }]
});
```

Puoi anche limitarlo a una parte dellÔÇÖapp con **`providers`** su un componente contenitore:

```typescript
@Component({
  selector: 'app-settings-shell',
  imports: [TailwindIcon],
  providers: [{ provide: TAILWIND_ICON_SIZE, useValue: 20 }],
  template: `<tailwind-icon icon="cog-6-tooth" />`
})
export class SettingsShellComponent {}
```

Passare esplicitamente **`[size]`** sulla singola icona ha sempre la precedenza sul valore del token.
